# 新功能实施计划

## 阶段一：阅读趋势图表功能 ✅ 进行中

### 目标
为用户提供可视化的阅读数据分析，包括：
- 每周/每月阅读趋势折线图
- 阅读热力图（类似 GitHub contribution graph）
- 阅读进度环形图
- 旧约 vs 新约阅读比例

### 技术方案
1. **安装图表库**: `recharts` (React 图表库)
2. **后端 API 扩展**: 添加数据聚合接口
3. **前端组件**: 创建图表组件
4. **页面集成**: 在 Home 页面或新建 Analytics 页面

### 实施步骤

#### Step 1: 安装依赖
```bash
cd client
npm install recharts
```

#### Step 2: 后端 - 添加数据聚合 API
- 文件: `server/src/controllers/reading.controller.ts`
- 新增接口:
  - `GET /api/reading/trends` - 获取阅读趋势数据
  - `GET /api/reading/heatmap` - 获取热力图数据
  - `GET /api/reading/progress` - 获取阅读进度统计

#### Step 3: 前端 - 创建图表组件
- `client/src/components/analytics/ReadingTrendChart.tsx` - 趋势折线图
- `client/src/components/analytics/ReadingHeatmap.tsx` - 阅读热力图
- `client/src/components/analytics/ProgressRing.tsx` - 进度环形图
- `client/src/components/analytics/TestamentPieChart.tsx` - 新旧约比例饼图

#### Step 4: 集成到页面
- 在 Home 页面添加"数据分析"区块
- 或创建独立的 Analytics 页面
